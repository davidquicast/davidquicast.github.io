PERSONA.md is the open specification for AI agent personhood: a declarative Markdown file with YAML frontmatter that defines an agent through ten layers, from identity and values to memory and metacognition.

The repository includes the spec, schema, CLI, validation, linting, export, diff, templates, and compiler targets for tools such as Claude Code and Codex.
