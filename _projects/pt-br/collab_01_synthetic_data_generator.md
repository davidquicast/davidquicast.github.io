**🤝 Colaboração no Ecossistema Hugging Face**

Contribuí com implementação Docker e Ollama para o projeto Synthetic Data Generator. A ferramenta containerizada usa Ollama para inferência de LLM local e Argilla para curação de dados, permitindo aos usuários construir datasets de alta qualidade usando descrições em linguagem natural.

### Contribuições Principais:
- Setup de containerização Docker
- Integração Ollama para inferência de LLM local
- Design de arquitetura para geração de dados containerizada
- Melhorias nas capacidades de deployment
